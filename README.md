# E-Shop

Simple online shop application.
The project was created to learn and test some libraries, frameworks and programming techniques.

## Technologies
* Java 9
* Maven 
* Spring
* JPA
* Vavr
* JUnit 5

## Features
* Product handling (create, modify)
* Bill handling (create, pay, cancel)
